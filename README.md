"Веб-сервис для продуктивности и тайм-менеджмента (инструмент типа To-Do / Task Manager)."

__Название проекта: TaskFlow__ (или DoNow, EasyTrack).

__Суть__: "Платформа для постановки, отслеживания и выполнения личных или командных задач."

## Цель и задачи системы

__Цель__: "Разработать удобный веб-интерфейс для управления задачами, который поможет пользователям структурировать дела, соблюдать дедлайны и повышать личную эффективность."

__Задачи системы:__

1) Обеспечить создание, редактирование и удаление задач (CRUD).

2) Реализовать систему категорий и приоритетов.

3) Научить систему фильтровать задачи по статусу (активные/завершенные).

4) Добавить поиск и сортировку.

5) Организовать хранение данных пользователя.

## Целевая аудитория

* Студенты (для учета домашних заданий и проектов).

* Фрилансеры и junior-специалисты (для контроля рабочих микрозадач).

* Любой человек, желающий вести список дел (списки покупок, личные цели).
